# Security and Data Handling

This artifact is synthetic-only and blind-review-only.

## Allowed inputs

- Invented screen text, invented speech fragments, invented event timing, and invented notification content.
- Small JSON fixtures under `data/synthetic/`.

## Disallowed inputs

- Live-session captures.
- Production logs or exports.
- Real user identifiers, access tokens, credentials, or internal links.
- Author or organization metadata while the submission is under anonymous review.

## Reporting issues

For the anonymous review version, record issues in the private issue tracker that accompanies this artifact. Do not include live data in reports.
